# Spawn Reference

## How Spawning Works

When a spawn command is triggered:

1. The targeted player receives the **auto-pull buff**, which periodically draws aggro from the spawned zombies.
2. Zombies are **randomly chosen** from the command's pool based on configured weights.
3. As **Game Stage** increases, base zombie counts will also increase.

## Extra Zombies via Super Chat

If your Super Chat donation exceeds the command's minimum price, you spawn **extra zombies** on top of the base count.

**Formula:** `extra zombies = floor((donation − minimum) ÷ threshold)`

- `donation` — your Super Chat amount in USD
- `minimum` — the command's Super Chat minimum (see table below)
- `threshold` — the USD increment per extra zombie (see table below)

**Example:** `!spawn normal` has a $1.00 minimum and a $0.25 threshold.
A $2.00 Super Chat → $1.00 over minimum ÷ $0.25 = **4 extra zombies** on top of the base 15.

> **Screamers** are an exception — they have no base count and no Super Chat scaling. The screamer horde size is determined entirely by your current Game Stage.
