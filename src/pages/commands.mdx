# Command Reference

All commands follow the syntax `!keyword @target` — see [How It Works → Command Syntax](/how-it-works#command-syntax) for full rules.

Costs are in points. The **Super Chat** column shows the minimum USD donation to trigger the command without spending points. Click any column header to sort.
